Entendendo Comunicação Client x Server

O professor em questão inicia o curso de uma maneira discursiva visando primeiro nos mostrar o começo da Web, nos mostrando os primeiros computadores como o ENIAC e o Harvard Mark I. Mostrando como é importante voltarmos no passado para utilizar as ferramentas atuais.

Uma breve e objetiva aula nos mostrando o que são Clients, relembrando como foi importante a demanda de consumo para que as tecnologias pudessem evoluir, como exemplo os navegadores, saindo do MOISAIC para o atual e poderoso Google.

Os clients tiveram uma grande parcela para o crescimento dos Servidores, que basicamente foi a inovação para a Tecnologia.

Introdução ao HTML na Prática

Explicação sobre os códigos utilizados para estruturar uma página web e todo seu conteúdo.

As tags são elementos utilizados para formatar e marcar um conteúdo em documento HTML, cada tag tem um nome e uma função, uma para links, texto em negrito, títulos e etc.

As listas são utilizadas para apresentar itens, cada uma com sua função. A lista ordenada enumera os elementos e a não ordenada acrescenta marcadores em cada elemento.

GitHub

A plataforma GitHub é uma plataforma de desenvolvimento utilizada para realizar projetos em nuvem, para que os desenvolvedores possam compartilhar e realizar mudanças em seus códigos de forma online.

É muito importante utilizar o GitHub para entregar os projetos, pois irá familiarizar os futuros desenvolvedores a utilizarem uma ferramenta que será muito importante para sua carreira, os acostumando e aprendendo como utilizar a plataforma.
